# SaaS Express

SaaS Express adds multi-tenancy to any application.

## Getting Started

```sh
docker run -ti --rm -p 8080:8080 \
  --platform linux/amd64 \
  quay.io/saasexpress/admin-api:latest
```

## Development

```sh
docker compose up
```

## Features

- Authentication
- Authorization
- Tenant management
- Audit logging
